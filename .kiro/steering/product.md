# Product Overview

## Aplikasi Unit Cost Rumah Sakit

A comprehensive hospital unit cost calculation system implementing Activity Based Costing (ABC) methodology with Double Distribution approach. The application manages cost allocation, revenue tracking, and financial analysis for hospital operations.

## Core Purpose

Calculate and analyze unit costs across 77+ hospital departments/units, enabling:
- Automated cost distribution using ABC methodology
- Real-time financial analysis and cost recovery tracking
- Medical procedure cost calculation (lab, radiology, surgery, etc.)
- Revenue vs cost analysis for hospital services
- Tariff scenario planning and budgeting

## Key Domain Concepts

- **Unit Kerja**: Hospital departments/work units (77 units total)
- **Distribusi Biaya**: Cost distribution across units (double distribution method)
- **Kalkulasi Biaya**: Cost calculation for various services (12 calculation modules)
- **Dasar Alokasi**: Allocation basis (SDM, room area, patient visits, etc.)
- **Jasa Pelayanan (JP)**: Service fees included in cost calculations
- **Cost Recovery**: Revenue to cost ratio analysis
- **Skenario Tarif**: Tariff scenarios for pricing strategy

## User Roles

- Super Admin, Admin, Manager, User, Guest
- Role-based access control with 280+ RLS policies
- Granular permissions for 100+ operations
