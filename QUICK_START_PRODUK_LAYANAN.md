# ⚡ Quick Start: Produk Layanan

## 🎯 Dalam 5 Menit

### Langkah 1: Buka Menu (10 detik)
```
Sidebar → 🛒 Produk Layanan
```

### Langkah 2: Tambah Data (1 menit)
```
1. Klik [+ Tambah Data]
2. Tab "Informasi Dasar":
   - Jenis: Rawat Jalan
   - LOS: 1
   - INA-CBG: A-4-10-I
   - Nama Dokter: Dr. Andi
```

### Langkah 3: Isi Diagnosa (30 detik)
```
Tab "Diagnosa & Prosedur":
   - Diagnosa 1: Hipertensi Esensial
   - Prosedur 1: Pemeriksaan Medis Umum
```

### Langkah 4: Tambah Layanan (2 menit)
```
Tab "Layanan":
   
   TINDAKAN:
   1. Klik [+ Tambah Tindakan]
   2. Pilih: T.001 - Konsultasi Dokter
   3. Qty: 1
   4. Klik [Tambah]
   
   LABORATORIUM:
   1. Klik [+ Tambah Laboratorium]
   2. Pilih: Lab.001 - Darah Lengkap
   3. Qty: 1
   4. Klik [Tambah]
```

### Langkah 5: Simpan (5 detik)
```
Klik [💾 Simpan]
✅ Data tersimpan dengan total biaya otomatis!
```

---

## 📦 Fitur Cepat

### Import CSV (30 detik)
```
1. Klik [↑ Import CSV]
2. Pilih file template_produk_layanan.csv
3. ✅ Data ter-import
4. Edit untuk tambah layanan
```

### Export CSV (10 detik)
```
1. Klik [↓ Export CSV]
2. ✅ File ter-download
```

### Edit Data (1 menit)
```
1. Klik [✏️] pada baris data
2. Ubah data yang diperlukan
3. Klik [💾 Simpan]
4. ✅ Data terupdate
```

### Hapus Data (5 detik)
```
1. Klik [🗑] pada baris data
2. Konfirmasi hapus
3. ✅ Data terhapus
```
